My Website Styling Guide

The recommended colors to use for the website are listed below:

Tumblr

#34526F

Youtube

#C4302B

DropBox

#3D9AE8

SoundCloud

#FF7700

The Tumblr color can be used for banners and title backgrounds.

The Youtube color should be used to draw attention to locations.

The Dropbox color should be used when you drop boxes.

The soundcloud color should be used for music.

The recommended text styles to use are as follows:

H1 Page heading/title

  • Font-Weight: 700 (bold)
  • Font-size: 70px
  • Font-Family: Lato

H2: Subheading

  • Font-Weight: 500
  • Font-size: 18px
  • Font-Family: Raleway

Paragraph

  • Font-Weight: Regular (400)
  • Font-size: 14px
  • Font-Family: Roboto
